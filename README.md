## The Signal

I treat technology as understood magic.

Scripts are spells. Exploits are alchemy. Linux is the grimoire. The terminal is the circle.

Not because tools are mystical, but because deep systems knowledge can feel impossible from the outside and obvious once decoded. My work lives in that threshold: breaking assumptions, building useful things, automating the boring parts, and documenting what the machine reveals.

I care more about curiosity than titles, more about reproducible knowledge than polished claims, and more about the craft than the noise around it.

## Operating Principles

| Principle | Translation |
| --- | --- |
| Knowledge before tools | A scanner is useful. Understanding the result is the actual skill. |
| Curiosity before certainty | Good research starts where easy answers stop working. |
| Build what you wish existed | Tools, notes, templates, labs, scripts, systems. |
| Document the path | A finding without context is a spark; a writeup is a torch. |
| Stay dangerous, stay ethical | Break systems only to understand, defend, improve, and teach. |
